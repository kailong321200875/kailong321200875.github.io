发表于 2019-02-10 分类于 npm 阅读次数：

在日常开发中，我们通常会通过npm下载各种各样的插件来为项目服务，这篇文章记录了一下如何编写一个插件并且上传到npm上，以vue组件为例。本篇文章省去了一些前置步骤，如如何创建npm账号，这些需要看官们自行去百度。
